# 🌦 Weather Prediction App

A Machine Learning project that predicts temperature based on environmental factors such as humidity, wind speed, and pressure. The project includes an interactive dashboard built with Streamlit.

## 🚀 Live Demo
Try it online: [Weather Prediction App](https://weatherprediction-aagg3dahbcirefbpkze3um.streamlit.app/)

## Features
- Interactive user interface with sliders for humidity, wind speed, and pressure
- Real-time temperature prediction using a Linear Regression model
- Dataset visualization with graphs

## Dataset
The project uses a CSV file (`data/data.csv`) containing:
- `temperature`
- `humidity`
- `wind_speed`
- `pressure`
